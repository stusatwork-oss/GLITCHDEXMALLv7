## FC-ARCADE (HARD COPY)

**Full Name:** HARD COPY - Arcade & Bookstore Hybrid
**Zone Type:** Commercial
**Status:** Liminal, Always Open

### Description

A dual-space anomaly. The arcade side glows with 80s neon - Pac-Man, Galaga, Asteroids, and one working Pong machine. Pink, blue, purple light. AOL dial-up era vibes. The floor is geometric patterned carpet with cigarette burns. Electronic beeps and boops fill the air.

Through a beaded curtain doorway, you enter the bookstore side. Deep purple walls. Tapestries with occult symbols. Worn carpet with incense ash. Shelves overstuffed with occult, new age, and conspiracy books. Dim lamps create a head-shop aesthetic. A lava lamp sits in the corner.

### Key Features
- **Beaded Curtain:** Click-clack as you pass between arcade and bookstore
- **Working Pong Cabinet:** Still functional, plays better than it should
- **Neon Lighting:** Never turns off, visible from corridor
- **Occult Books:** New age, conspiracy, liminal space theory

### Artifacts
- Arcade tokens (never sold, only given as rewards)
- Beaded curtain strand (fell on its own)
- Occult bookmark (from a book that was never here)
- AOL Trial CD (1997, account still exists)
- Pong paddle grip (fell off but paddle plays better without it)

### Vibe Flags
- Liminal
- Nostalgia
- Occult
- Gaming
- 80s/90s aesthetic

### Notes
"That HARD COPY place gives me the creeps. The neon never turns off." - Milo

## STORE_BORED

**Full Name:** BORED - Skateshop & Tee Emporium
**Zone Type:** Youth Culture
**Status:** Technically Operating (Expired Lease)

### Description

A cramped skateshop run by a 17-year-old named BORED. Walls plastered with band posters, graffiti tags, and layers of stickers from decades of mall kids. Skate decks lean against black-painted concrete walls. The floor is smoothed by wear and skateboard wax. Air smells like grip tape and teenage ennui.

The counter overflows with tees, hoodie stacks (200+ hoodies from dad's last order), and unboxed skate parts. Functional lighting that's moody by accident.

### Key Features
- **Band Posters:** Covers every surface
- **Graffiti:** Half the mall's tags came from markers sold here
- **Skate Parts:** Trucks, bearings, grip tape, decks
- **Hoodie Mountain:** 200 hoodies, still working through them

### Artifacts

- Graffiti marker (silver ink, almost empty)
- Torn band poster (STATIC VOID, 1999 tour)

### NPCs
- BORED (owner/operator, inherited from dad)

### Vibe Flags
- Youth culture
- Minor chaos
- Posters everywhere
- Secret hint source
- Skate energy

### Notes
"Everyone shops online now." - BORED
"Dad ordered like 200 hoodies before he left. Still working through them." - BORED

## COMPHUT

**Full Name:** COMPHUT
**Zone Type:** Retail (Electronics/Media)
**Status:** Abandoned (No Staff)

### Description

A Blockbuster/RadioShack abomination. Yellow and blue corporate colors, faded. Scuffed linoleum floors. Half the store is VHS tapes in plastic cases ("Be Kind Rewind"). The other half is transistors, cables, resistors in plastic bins, and obsolete computer parts.

Shelves are semi-filled. The cash register is from 1987, covered in dust. Some fluorescent tubes are dead. No workers anywhere. No customers in weeks.

### Key Features
- **VHS Section:** Action, comedy, horror tapes
- **Electronics Section:** Cables, adapters, dead components
- **Abandoned Counter:** 1987 register, dusty
- **Corporate Decay:** Faded branding, dying lights

- Dead AA battery (corroded, from a 1994 Walkman)

## ANCHOR_STORE

**Full Name:** The Anchor Store (Name Forgotten)
**Zone Type:** Department Store (Large)
**Status:** Dead, Liminal

### Description

A massive, dead department store. Mannequins stand frozen in dark windows. Most lights are out. Exposed black ceiling grid shows infrastructure. Empty display cases. Clearance racks with nothing on them. A lone vending machine hums in the darkness.

Echoes carry strangely here. Your footsteps sound wrong. The space is huge and empty and cold. This is where retail comes to die.

- **Exposed Ceiling:** Black grid, infrastructure visible

## FOOD_COURT

**Zone Type:** Dining (Below Grade)
**Status:** Mostly Closed (Slushee Machine Still Works)

Natural light filters down mixed with dying fluorescent glow. The slushee machine hums. It's the only thing that still works.

## Integration Notes for V6

### Cloud Level Mappings

V3 used "Heat" (0-5), V6 uses "Cloud" (0-100). Here's the mapping:

| V3 Heat | V6 Cloud | Mood | Description |
|---------|----------|------|-------------|
| 0-1 | 0-24 | Calm | Subtle weirdness, almost normal |
| 2 | 25-49 | Uneasy | Skyrim-level dissonance, schedule talk |
| 3 | 50-74 | Strained | Mask slipping, glitches visible |
| 4 | 75-89 | Critical | AI exposure, reality breaks |
| 5 | 90-100 | Bleed | Full simulation reveal |

### Zone Microstate Enrichment

Use these descriptions to populate:
- `zone.description` - Atmospheric detail
- `zone.features` - Interactive elements
- `zone.artifacts` - Discoverable lore items
- `zone.npc_spawns` - Character locations
- `zone.vibe_tags` - Mood/aesthetic flags

### Artifact Discovery System

Implement discovery events in UE5:
```json
{
  "type": "discover",
  "document": "arcade_token",
  "zone": "FC-ARCADE"
}
```

Send to bridge when player finds artifacts. Cloud reacts to discoveries.

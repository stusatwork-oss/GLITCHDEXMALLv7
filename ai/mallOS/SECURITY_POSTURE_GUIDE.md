## üßø 7. UNIVERSAL SECURITY POSTURE RULES

All ranks obey these posture laws:

### 1. No weapons. Ever.
Hands remain empty, open, or on gear.

### 2. Movement = identity.
Silhouette must stay consistent with rank.

### 3. No sudden flinches.
Surprise is expressed through:
- posture tightening
- gaze sharpening

Never cartoonish motion.

### 4. Synchronicity matters.
If two guards walk together, their timing subtly matches.

### 5. Posture predicts Cloud.
- Upright = Cloud dampens
- Hunched = Cloud thickens
- Scanning = Cloud volatile
- Stillness = emotional compression

### 6. Hierarchy visible through distance.
- Execs walk side-by-side
- Sergeant stays ahead of New Guy
- Fox circles perimeter
- Raven stays alone

### 7. Tailoring = rank.
Better tailoring ‚Üí more composed posture.
